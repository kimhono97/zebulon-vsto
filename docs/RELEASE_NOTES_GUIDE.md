# 릴리스 노트 작성 지침 (Release Notes Guide)

ZebulonVSTO 릴리스 노트(주로 GitHub Releases에 게시)를 쓸 때의 표준입니다.

> 요약: **"무엇이 달라졌나"를 사용자 관점에서, 분류해서, 짧게.** 설치·요구사항 같은 고정 정보는 매번 전문을 반복하지 말고 `deploy/README.txt`를 가리킨다. P2P 도구이므로 **호환성·업그레이드 영향**은 항상 명시한다.

## 게시 위치 / 버전·태그
- 게시: GitHub Releases (`releases/new?tag=v<버전>`), zip 자산 첨부(`ZebulonVSTO-<버전>.zip`).
- 버전: 4자리 `주.부.빌드.수정`, 태그는 `v` 접두사(예: `v1.0.1.0`).
- (선택) 누적 기록이 필요하면 저장소에 `CHANGELOG.md`를 두고 같은 항목을 역순으로 쌓는다.

## 원칙
1. **사용자 관점** — 내부 구현이 아니라 사용자가 보거나 하는 변화로 쓴다. (O "자동 탐색 추가" / X "DiscoveryResponder 클래스 추가")
2. **분류** — 새 기능 / 개선 / 버그 수정. 해당 없는 분류는 생략.
3. **고정 정보는 포인터로** — 설치·요구사항은 거의 안 바뀌므로 `README.txt`를 가리키고 "이번에 달라진 점"만 적는다.
4. **호환성·업그레이드 영향 명시** — 송·수신 상호운용, 포트/프로토콜 변경, 깨지는 변경, "양쪽 업그레이드 필요" 등.
5. **짧게 + 링크** — 자세한 기술 내용은 PR/사양서 링크로 넘긴다.
6. **언어** — 한국어(최종 사용자 대상). 코드 식별자·명령·파일명은 원문 그대로.

## 템플릿
```markdown
## v<버전> (<YYYY-MM-DD>)
<한 줄 요약>

### 주요 변경
- **새 기능**: …
- **개선**: …
- **버그 수정**: …

### 호환성 / 업그레이드
- <혼재 동작·포트/프로토콜·깨지는 변경. 없으면 "기존과 호환, 특이사항 없음".>

### 설치 / 업데이트
- 신규: 압축 해제 후 `Install.ps1` 실행 — 자세히는 `README.txt`.
- 업데이트: 새 zip을 받아 PowerPoint를 닫고 `Install.ps1` 재실행(덮어쓰기).
- 요구사항(바뀐 경우에만 언급): PowerPoint 2013+, VSTO 2010 런타임.

### 알려진 문제 (있을 때만)
- <증상 + 회피책>

### 다운로드
- `ZebulonVSTO-<버전>.zip`
```

## 과거 형식(설치 / 요구사항 / 주요 변경)과의 차이
- **주요 변경**은 유지하되 **새 기능 / 개선 / 버그 수정**으로 분류.
- **설치·요구사항**은 매번 전문을 반복하지 않고 **README 포인터 + 변경분만** → 노트가 짧아지고 중복이 준다.
- **호환성 / 업그레이드**, **알려진 문제** 섹션을 추가(P2P라 버전 혼재 영향이 중요).

## 예시 — v1.0.1.0
```markdown
## v1.0.1.0 (2026-06-29)
피어 자동 탐색과 설정 마법사를 추가하고 리본을 개편했습니다.

### 주요 변경
- **새 기능**: 송신 설정에 "자동으로 찾기" 추가 — 같은 서브넷의 수신기를 자동 인식해 목록에서 선택(전체 브로드캐스트 또는 1대 지정). 수동 입력도 그대로 유지.
- **개선**: 시작 시 설정 마법사 도입(역할 → 수동/자동). 리본은 시작/중지 + 실시간 자기/상대 상태(IP·포트·역할) 표시로 단순화.
- **개선**: 진단 도구(Tools)에 Probe / Responder 모드와 대화형 입력 추가.

### 호환성 / 업그레이드
- 자동 탐색은 **송·수신 양쪽이 1.0.1.0 이상 + 같은 서브넷**일 때만 동작(탐색용 UDP 포트 8290 추가). 안 되면 **수동 입력**을 사용(버전 혼재에서도 동작).

### 설치 / 업데이트
- 신규: 압축 해제 후 `Install.ps1` (자세히는 `README.txt`).
- 업데이트: 새 zip을 받아 PowerPoint를 닫고 `Install.ps1` 재실행.

### 다운로드
- `ZebulonVSTO-1.0.1.0.zip`
```
